Some special tags of HTML

1. HyperLink

    For Text HyperLink
  • < a href="location_with_name.html" >Any_text< /a >= It is used to indicate the Link destination like to open facebook Click here or to open Lecture-1 Click here.
    • For Image hyperLink
  • < a href="location_with_name.html" >< img src="location_with_name.image_format" >< /img >< /a >= It is used to indicate the link destination on image like

2. Image Inserting

  • < img src="location_with_name.image_format"> = It is used to insert any image from drive like

3. Audio Inserting

  • < audio controls >< source src="location_with_name.audio_format" type="audio/audio_format" >< /audio >= It is used to insert audio from drive like
    • For AutoPlay
  • < audio controls autoplay >< source src="location_with_name.audio_format" type="audio/audio_format" >< /audio >= It is used to insert audio from drive and play automatically like
    • For autoplay and Repeat
  • < audio controls loop >< source src="location_with_name.audio_format" type="audio/audio_format" >< /audio >= It is used to insert audio from drive and repeat song like

4. Video Inserting

  • < video controls >< source src="location_with_name.video_format" type="video/video_format" >< /video >= It is used to insert video from drive like
    • For AutoPlay
  • < video controls autoplay >< source src="location_with_name.video_format" type="video/video_format" >< /video >= It is used to insert video from drive and play automatically like
    • For autoplay and Repeat
  • < video controls loop >< source src="location_with_name.video_format" type="video/video_format" >< /video >= It is used to insert video from drive and repeat video like

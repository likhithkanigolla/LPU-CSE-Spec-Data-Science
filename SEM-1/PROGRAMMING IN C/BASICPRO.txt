5)INCREMENT/DECREMENT OPERATOR:-

	These are used to control the loops in an efficient method.

  INCREMENT OPERATOR:-

	The symbol ++ is used for increment by 1

	++identifier  컴컴컴->prefix increment

	identifier++  컴컴컴>postfix increment

	eg:-
	int n=10;
	++n (or) n++;
	n=11;

prefix:-
int a=10,b;
b=++a;
a 컴컴>11
b 컴컴>11

postfix:-

int a=10,b;
b=a++;
a 컴컴>11
b 컴컴>10

#include<conio.h>

{
int i=10;
clrscr();
printf("%d\t%d",i,i++);
getch();
}

SPECIAL OPERATOR:-

A)TERINARY OPERATOR         (OR)
  CONDITIONAL OPERATOR:-

	A terinary operator pair ? : is available in the c language .It is
  user to construct a conditional expression.

	The general form of the terinary operator:

	exp1?exp2:exp3;

		If exp1 is true then exp2 is evaluated and its value becomes
  value of the expression other wise exp1 is flase then exp3 is evaluated
  and its value becomes value of expression.

  eg:-
  int a=20,b=10,max;

  max=a>b?a:b;
  a=20;
  b=10;
  max=20;



  eg2:-

  int a=20,b=10,min;
  min=a<b?a:b;

  a=20;
  b=10;
  min=10;

  WRITE A PROGRAMME TO MAX AND MIN VALUES OF GIVEN TWO NUMBERS USING

COMMA OPERATOR:-

	It can be used the link the related expressions together the general
  form of the comma operator is:-

  eg:-
  int a,b,c;
  c=(a=10,b=20,a+b);
  a=10;
  b=20;
  c=30;

  In the above example first assigns the value of 10 to a then 20 to b
  and finally assigns 30 to c(a+b)

  type casting.

	In c language type conversion is a arithmatic conversion will be done
  automatically this is called Implicit conversion.

	If you want store a value of one type into a variable of another type
  we must caste the value to be stored by preceeding with type name in
  parenthasis.


  int a,b;
  float c,d;

  a=5;
  b=2;

  c=a/b;      implicit conversion   or   arithmatic conversion

  c=2.00

  d=(float)a/b;  explicit conversion or type casting

  d=2.50

				 ARRAYS
	An arrays is a collection of data items of same data types stored
   in a continuous memory location and it can be refered as a common
   name.A particualr value is indicated by writing number called index or
   subscript in square brasis after the array name

   TYPES OF ARRAYS:-
   C supports 3 types of arrays they are

   1)SINGLE DIMENSIONAL ARRAY(ONE DIMENSIONAL ARRAY)
   2)DOUBLE DIMENSIONAL ARRAY(TWO DIMENSIONAL ARRAY)
   3)MULTY DIMENSIONAL ARRAY


1)SINGLE DIMENSIONAL ARRAY:-
      A list of items can be given one variable name using only one index.
  Such a variable is called as single dimensional array.A single dimensioanal
  arrays the elements are represented one after to the other in edges of
  memory bytes.

  decleration:-       datatype    arr_name[size];
   eg:-            int  a[5];

  elements:- a[0],a[1],a[2],a[3],a[4]

  NOTE:-In any array the array index from 0 to size-1.

INITIALISATION:-

datatype arr_name[size]={value-1,value-2,---------,value-n};
		   |
		  optional

 eg:- int a[5]={1,2,3,4,5};
      int arr[]={100,200,300};

      This is the initialisation.

TWO DIMENSIONAL ARRAYS:-

Two dimensional arrays can store a table of values which contains rows and
cols.We can use one variable name and two index values the first index is
for rows and the second index is for columns.

decleration:-

datatype arr_name[row size][col size];

eg:-

int a [2][3]             size is 12 bytes

elements:-

a[0][0]     a[0][1]      a[0][2]
a[1][0]     a[1][1]      a[1][2]

Initialization:-

form-1:-
datatype arr_name[row size][col size]={val-1,val-2,----val-n};

ex:-
int a[3][3]={1,2,3,4,5,6,7,8,9};
int a [][3]={1,2,3,4,5,6};

form-2:-
datatype arr_name[row size][col size]={ {val-1,val-2,------,val-n},
				       {val-1,val-2,------,val-n},
				       ----------------------------
				       ----------------------------,
				      {val-1,val-2,------,val-n} };

Ex:-

int a[3[3]={ {1,2,3},
	     {4,5,6},
	     {7,8,9} };

int checkmatrix(int a[][20],int m,int n)
{
int i,j;
for(i=0;i<m;i++)
{
for(j=0;j<n;j++)
{
if(i==j)
{
if(a[i][j]!=1)
return 0;
}
else
{
if(a[i][j]!=0)
return 0;
}
}
}
return 1;
}

void multiplication(int a[][20],int b[][20],int c[][20],int m,int n,int q)
{
int i,j,k;
for(i=0;i<m;i++)
{
for(j=0;j<q;j++)
{
c[i][j]=0;
for(k=0;k<n;k++)
{
c[i][j]=c[i][j]+a[i][k]*b[k][j];
}
}
}
}






MULTI DIMENSIONAL ARRAYS:-

C supports arrays are three or more dimensional .In multi dimensional
arrays we can use more than two index values.The general form of multi
dimensional array is

syntax:-
datatype arr_name[s1][s2]-----[sn];

eg:-
int a[2][2][3];

elements:-

a[0][0][0]		a[0][0][1]		a[0][0][2]
a[0][1][0]		a[0][1][1]		a[0][1][2]

a[1][0][0]		a[1][0][1]		a[1][0][2]
a[1][1][0]		a[1][1][1]		a[1][1][2]

		   CONTROL STATEMENTS
			  OR
		    CONTROL STRUCTURES

C is considered as a structured programming language one of the
  reason for this is having various programme control statement.C process
  the decision making capabilities and support the statements known  as
  control statements. There are three types of control statements supported
  by C.

  1)CONDITIONAL CONTROL STATEMENTS
  2)UN-CONDITIONAL CONTROL STATEMENTS
  3)LOOP CONTROL STATEMENTS

  1)CONDITIONAL CONTROL STATEMENTS:-

	C supports five types conditional control statements.There are

	1)simple if statement
	2)if else statement
	3)nested if statement
	4)else if ladder statement
	5)switch statement


	1)simple if statement:-It is a decesion making statement and used
  to control the flow of execution.

  syn:- if(expr)
	{
	statements;
	}

	If the expression is true that the statement is executed and the
  control transfer the next statement otherwise expression is false then
  the control directly go to the next statement.

if else STATEMENT:-
	It is an extension of simple if statement.
   syn:-

   if(expr)
   {
   statements-1;
   }
    else
   {
   statements-2;
   }
	  If the expression is true then if block statements are executed
  and else block statement are ignore.Ohterwise expression is false then
  else block statement are executed and if block statement are ignore.

else if ladder statement:-

This statement is also used for the series of decesions are involved.

syn:-

   if(expr-1)
   {
   statment  -1;
   }
   else if(expr -2)
   {
   ---------------
   ----------------
   }
   else if (expr -n)
   {
   statements -n;
   }
   else
   {
   statment -else block
   }


	In the statement expression are evaluated from top to bottom.
  If the condition is true the statements associated the block is executed
  and the control transfer to the next statement.When all expressions are
  false finally else block statements will be executed.

SWITCH STATEMENT:-

It is a multiway conditional statement using c language.It is mainly used in
situations where there is needed to pick one alternative among many
alternatives.

syntax:-

switch(exp)
{
case value1:
	statements-1;
	break;
case value2:
	statements-2;
	break;
	-----------
	-----------
case value-n:
	statements-n;
	break;
default:
	default statements;
	break;
}

	The switch statement tests the value of given variable or expression
  agains a list of case values.When a match is found,a block of statements
  associated the case is executed.Otherwise the default block statement is
  will be executed.

NOTE 1:-Switch statement the value of expression or variable is integer
  (int,char)are cannot pass string and floating values.

NOTE 2:In switch statement the default block is optional.

break;
  It is a unconditional control statement, it terminates a switch statement
  and a loop statment

  syn:-
  break;

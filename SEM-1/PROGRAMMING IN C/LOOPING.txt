
			     LOOPING STATEMENTS

	The process of repeated executing a block of statements up to
   specified number of times is called as loop.C supports three types of
   looping statements.

   1)while loop
   2)do while loop
   3)for loop

 Any loop as three things are

	   1)Initialize the index
	   2)test condition
	   3)update the index

while loop:-
	It is a conditional control loop statement in 'C' language.

syntax:-
     while(test condition)
     {
       statements;
      }


	In this statement first the test condition will be evaluated,if it
  is true then the statement block will be executed.After the execution
  of statements the test condition once again evaluated.If it is true then
  the statement block will be executed once again.This process of repeated
  execution continuous untill the test condition finally becomes false.

DO-WHILE LOOP:----

	It is a alternative form of while loop.The only difference between
  while and do while is the minimum number of execution of while is zero
  and the minimum number of execution of do while is one.

  syn:-
  do
  {
  statements;
  }
  while(test condition);

	First the statement block will be executed and then the text condition
  is executed and then the text condition is evaluated.If it is true then the
  statement block will be executed once again.This process of repeated
  execution continuous and till the text condition becomes false.

for loop:-
	It is the most commonly used loop statement in c language.It
  consisting of three expressions.

  syn:-
  for(exp1;exp2;exp3)
  {
  statements;
  }
	The first expression is used to initialise the index,the second
  expressionis used to check whether the loop is to be continued or not
  (test condition),and the third expression is used to change the index for
  further iteration.
